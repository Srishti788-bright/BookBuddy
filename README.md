# 📚 BookBuddy

BookBuddy is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application that helps users organize their personal book collection. It allows users to add, search, and delete books while also providing a Reading Planner with an interactive calendar to track reading progress.

---

## 🚀 Features

### 📖 Library
- ➕ Add new books
- 🗑️ Delete books
- 🔍 Search books instantly
- 📚 View all saved books

### 📅 Reading Planner
- Interactive calendar
- Click on a date to mark it as a reading day
- Selected dates are highlighted in dark red
- Clean and responsive interface

### 🏠 Home Page
- Modern landing page
- Motivational reading quotes
- Feature cards
- Deep blue professional UI

### 📄 Additional Pages
- About
- Contact
- Reviews

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Srishti788-bright/BookBuddy.git
```

### 2. Go to the project folder

```bash
cd BookBuddy
```

### 3. Install Backend Dependencies

```bash
cd backend
npm install
```

### 4. Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

### 5. Create a .env file inside the backend folder

```env
MONGO_URI=your_mongodb_connection_string
```

### 6. Start Backend

```bash
cd backend
node server.js
```

### 7. Start Frontend

```bash
cd frontend
npm run dev
```
